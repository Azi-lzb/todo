# 工作任务清单 · 本地版

完全本地运行，无需部署、无需联网。数据保存在浏览器本地（localStorage），不会上传到任何服务器。

---

## 使用方式

用浏览器直接打开 `index.html`（双击或在浏览器中打开该文件即可）。建议使用 Chrome、Edge、Firefox 等现代浏览器。

---

## 功能概览

### 1. 工作台

- **月历视图**：顶部选择年份、月份，以月历形式展示整月；每个日期格内显示当日任务，底部有「整月任务（未指定日期）」区域。
- **今日任务侧栏**：右侧固定显示「今日」任务列表；下方有统计：今日共 X 个任务、已完成 X 个、剩余 X 个。
- **刷新 / 跳到今天**：工具栏中年份、月份旁的圆形箭头按钮，点击后切到当前日期所在年月并刷新视图（便于跨天后更新「今日任务」）。
- **新建任务**：在工作台点击「新建任务」，填写任务名称、日期（可选，不选则为整月任务）、轻重缓急、操作指南、注意事项、操作步骤（可不填，支持 0 步）；保存与关闭在弹窗右上角。
- **从高频任务部署**：点击「从高频任务部署」，选择年份月份、勾选要部署的高频任务，再勾选「整月」或点击月历选择具体日期，确认后任务会出现在对应月份。

### 2. 任务详情与编辑

- 点击任意任务可进入详情：查看操作指南、注意事项、操作步骤；可进行**编辑**、**归档（已完成）**、**删除**。
- 编辑时可将当前任务**保存/同步到高频任务库**：若该任务原本由高频任务部署，会更新原高频任务；否则在高频库中新建一条模板。
- 操作步骤可为空，不添加步骤即表示无步骤。

### 3. 高频任务库

- 切换至「高频任务库」标签：保存常用任务模板（名称、操作指南、注意事项、操作步骤，步骤可为空），便于后续一键部署到某月某天。
- 每条高频任务可**编辑**、**部署到某月**、**删除**；点击任务名称可查看详情。

### 4. 今日侧栏 · 备注

- 在「今日任务」与统计信息下方有**备注**区域。
- 点击「备注」标题旁的 **+** 可新增一条备注；每条备注支持**编辑**（铅笔图标）、**保存**（勾选图标）、**删除**（垃圾桶图标），仅图标操作无文字按钮。
- 备注数据随导出/导入一并备份。

### 5. 数据与清理（右上角下拉）

点击右上角「数据与清理 ▾」展开菜单：

- **导出全部数据**：下载 `todolist-data.json`（含任务、高频任务、备注），可拷贝到项目根目录或他机备份。
- **导入数据文件**：选择此前导出的 JSON 文件，覆盖当前任务、高频任务与备注。
- **清空当前月任务**：仅清空当前所选年月的任务。
- **按日期清空某天任务**：按提示输入年、月、日，清空该日任务（不影响整月任务）。
- **清空高频任务库**：仅清空高频任务模板。
- **全部清空（任务 + 高频）**：清空所有月份任务及高频任务库（最终手段）。

## 典型流程

1. **建高频任务**：在「高频任务库」中「添加高频任务」，填写名称、操作指南、注意事项与步骤（可不填），保存。
2. **部署到某月**：在工作台选好年月，点「从高频任务部署」，勾选任务与目标日期或「整月」，确认部署。
3. **日常使用**：在工作台查看月历与今日任务，点击任务查看详情或编辑；完成后在详情中「归档」；需要时在备注区写当日提醒。
4. **备份与移植**：定期用「导出全部数据」保存 JSON，换设备或重装后用「导入数据文件」恢复。

## 文件说明
